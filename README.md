# ☕ Meus Projetos em Java

> Repositório focado no aprendizado e aplicação prática de Programação Orientada a Objetos (POO), arquitetura de software e resolução de problemas reais em Java.

---

## 📁 Projetos Incluídos

### 📚 [Catalogador de Biblioteca Universitária](./Catalogador%20de%20Biblioteca%20Universitaria)
Sistema para cadastro, classificação por áreas do conhecimento e consulta dinâmica de obras acadêmicas.
- **Foco:** Abstração, Coleções (`List`/`ArrayList`) e Enums.

### 📊 [Engine de Risco de Crédito](./Engine%20de%20Risco%20de%20Credito%20(POO%20Avançada))
Motor de análise financeira que calcula *credit score*, define limites e aplica taxas de juros com base no perfil do cliente.
- **Foco:** POO Avançada, Tratamento de Exceções e Design Patterns (*Strategy*).

### 🏧 [Simulador de Caixa Eletrônico](./Simulador%20de%20Caixa%20Eletronico)
Aplicação interativa para operações bancárias como saques com distribuição otimizada de cédulas, depósitos e extratos.
- **Foco:** Lógica de negócios, validações de saldo e controle de fluxo.
